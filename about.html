About the publication

Built for architects
who work with AI.

ArchiGen is an editorial platform covering AI tools, workflows, and techniques for architectural visualization. We test software, document real workflows, and write honestly about what actually works in studio practice.

Editorial opinions are our own. Every tool we cover gets run through a real project before we write about it — our reviews reflect actual studio practice, not vendor marketing.

Read the journal → Browse workflows

What we cover

The full pipeline from concept sketch to client-ready render — and every AI tool that touches it. We focus on what's actually useful to architects and archviz artists working under real deadlines with real clients.

That means render engines, concept generators, upscalers, inpainting workflows, ComfyUI setups, prompt engineering for architectural subjects, and the broader question of how these tools change how studios are run.

What we don't do

  • Reviews driven by vendor relationships — if a tool doesn't hold up in practice, we say so
  • Benchmarks that ignore real studio constraints (budget, iteration speed, client expectations)
  • Hype cycles — we wait until we've actually run the tool through a project
  • Tutorials that assume a render farm and unlimited compute time
